# changelog

## 4.0.6 - 2026-07-06

стабилизационный патч миграции и системных настроек.

- конфликтующие старые шаблоны прицелов теперь сохраняются как `*.legacy.json` и отображаются как `Имя (legacy)`.
- автозапуск после переключения сразу сохраняет только свое состояние, не затрагивая другие несохраненные настройки.
- мониторинг разрешения экрана корректно останавливается при выходе из приложения.
- горячая клавиша при удержании больше не спамит включение и выключение прицела; одно нажатие дает одно переключение.

## 4.0.5 - 2026-07-06

патч проверки обновлений.

- если GitHub API недоступен или уперся в rate limit, проверка обновлений использует fallback через страницу `releases/latest`.
- ссылка на установщик строится через стабильный путь `releases/latest/download/CrossHud_Setup.exe`.
- API остается основным способом проверки, fallback включается только при ошибке.

## 4.0.4 - 2026-07-06

патч автозапуска после переименования приложения.

- если автозапуск включен в настройках, при запуске приложение пересоздает ключ реестра `CrossHud`.
- старые ключи автозапуска `CrossHud_PetyaBlatnoy`, `CrossHud_By_PetyaBlatnoy`, `Crosshud_By_Petyablatnoy` и `Crosshud_By_Petyablatnoy-HudsightKiller` удаляются.
- если автозапуск выключен в настройках, новый ключ не создается.

## 4.0.3 - 2026-07-05

патч ярлыков и очистки старых папок после переименования в `CrossHud`.

- ярлык меню пуск создается в пользовательском меню пуск по пути `AppData\Roaming\Microsoft\Windows\Start Menu\Programs`.
- установщик удаляет старые ярлыки `CrossHud_By_PetyaBlatnoy`, `Crosshud_By_Petyablatnoy` и `Crosshud_By_Petyablatnoy-HudsightKiller`.
- старые папки данных в профиле пользователя, `AppData\Local` и `AppData\Roaming` переносятся в `CrossHud`.
- старые папки данных удаляются только после успешного переноса.
- если в `CrossHud` уже есть свои настройки, старый конфликтующий `settings.json` сохраняется как legacy-копия.
- старые папки установки в `Program Files` удаляются при установке новой версии.

## 4.0.2 - 2026-07-05

патч миграции после переименования приложения.

- настройки и шаблоны из `CrossHud_By_PetyaBlatnoy` теперь подхватываются даже если новая папка `CrossHud` уже создана логами.
- если новая папка содержит только дефолтные настройки, они заменяются старыми пользовательскими настройками.
- старый uid переносится вместе с настройками.
- старые шаблоны копируются в новую папку без перезаписи уже существующих файлов.
- установщик удаляет старый ярлык меню пуск `Crosshud_By_Petyablatnoy-HudsightKiller`.

## 4.0.1 - 2026-07-05

патч переименования репозитория и приложения.

- приложение переименовано в `CrossHud`.
- установщик теперь называется `CrossHud_Setup.exe`.
- настройки и шаблоны из старой папки `CrossHud_By_PetyaBlatnoy` переносятся в новую папку `CrossHud` при первом запуске.
- при тихом обновлении установщик запускает новую версию после установки.
