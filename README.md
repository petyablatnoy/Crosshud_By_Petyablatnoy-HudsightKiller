# crosshud v3 (by petyablatnoy)

**бесплатный опенсорсный убийца hudsight.**

crosshud v3 — это полная переработка предыдущей версии. теперь это полноценное приложение с современным дизайном, максимальной производительностью и функционалом, который наравне с платным hudsight оставаясь полностью бесплатным и с открытым исходным кодом.

зачем платить за то, что можно получить бесплатно? crosshud — гибкий прицел-оверлей для windows, который дает все возможности платных аналогов.

##  почему crosshud, а не hudsight?

| фича | crosshud (этот проект) | hudsight (платный аналог) |
| :--- | :---: | :---: |
| **цена** | **бесплатно** | платно |
| **открытый код** | **да** | нет |
| **пиксельный редактор** | **да** | да |
| **настройка прицела** | полная | полная |
| **шаблоны/профили** | **да** | да |
| **безопасность** | **полная прозрачность** | закрытый исходный код |

##  для пользователей

### возможности v3

*   **работает везде:** благодаря новой системе цифровой подписи, прицел отображается поверх любых окон, включая игры в **эксклюзивном полноэкранном режиме** и поверх приложений с правами администратора.
*   **современный дизайн:** полностью новый интерфейс в темном стиле discord-style с плавными анимациями.
*   **кастомный  редактор:** рисуйте свой прицел по пикселям. теперь с **зумом (ctrl+колесо)** и **перемещением (средняя кнопка мыши)**.
*   **гибкая настройка:** размер, толщина, зазор, прозрачность, обводка, центральная точка.
*   **rgb эффекты:** режим радуги.
*   **умное поведение:**
    *   скрытие при зажиме пкм.
    *   скрытие по горячей клавише (по умолчанию `insert`).
    *   автозапуск вместе с windows.
    *   запуск сразу свернутым в трее.

### 📥 установка и запуск

1.  **скачать:** загрузите установочный файл `CrossHud_By_PetyaBlatnoy_Setup.exe` из раздела релизов.
2.  **установить:** запустите установщик.
    *    **важно:** в процессе установки программа автоматически установит специальный сертификат. **это необходимо**, чтобы прицел мог работать поверх игр с античитами и в полноэкранных режимах. не пугайтесь, это безопасно и нужно для функционала.
3.  **играть:** запустите crosshud, настройте прицел под себя и запускайте игру.

### управление
*   **вкл/выкл прицел:** переключатель в меню или горячая клавиша (по умолчанию `insert`).
*   **свернуть:** приложение закрывается в системный трей, чтобы не мешать.

##  для разработчиков

если вы хотите разобраться в коде или собрать свою версию, вот как устроена прога.

### технический стек
*   **язык:** python 3.10+
*   **ui фреймворк:** **pyside6 (qt)** — *отказ от tkinter в пользу красивого ui.*
*   **графика:** **pillow (pil)** — *для отрисовки прицела в память.*
*   **система:** **winapi (ctypes, pywin32)** — *для создания прозрачного окна (layered window) и хуков.*

### структура проекта
v3 имеет модульную архитектуру:

1.  **`main_file.py`**: точка входа. проверяет **mutex** (защита от повторного запуска), настраивает логирование и запускает qt приложение.
2.  **`crosshud_app.py`**: главный контроллер. связывает ui, оверлей и настройки через систему сигналов (signals & slots).
3.  **`ui_components.py` & `qt_widgets.py`**:
    *   здесь происходит вся магия интерфейса.
    *   реализована кастомная верстка без стандартных заголовков windows.
    *   собственные виджеты: `discordswitch`, `modernslider`, `settingcard`.
4.  **`overlay_manager.py`** (сердце программы):
    *   создает прозрачное окно через `ws_ex_layered | ws_ex_transparent`.
    *   **рендеринг:** в отличие от v2 (где использовался медленный `setpixel`), здесь прицел рисуется библиотекой **pil** в буфер памяти, а затем мгновенно передается в windows через `updatelayeredwindow`. это дает идеальную прозрачность для курсора и скорость.
    *   содержит хуки для отслеживания мыши (пкм) и клавиатуры.
5.  **`settings_manager.py`**:
    *   управляет `settings.json`.
    *   **авто-миграция:** при первом запуске умеет подхватывать старый конфиг от v2.
6.  **`pixel_editor.py`**: виджет редактора с поддержкой масштабирования и сетки.

### сборка (build)
используется скрипт `build.py`, который выполняет 3 шага:
1.  **сборка exe:** через pyinstaller с флагом `--uac-uiaccess` (критично для работы поверх админских окон).
2.  **подпись кода (`sign_app.ps1`):** создает самоподписанный сертификат и подписывает exe файл. без этого windows не даст приложению рисовать поверх "сильных" окон.
3.  **создание установщика:** компиляция nsis скрипта, который также устанавливает сертификат в доверенные при установке.
